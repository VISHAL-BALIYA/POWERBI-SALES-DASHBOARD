# FMCG Sales Performance Dashboard (Power BI)  ## 📊 Project Overview This repository contains my first comprehensive Power BI business intelligence report. The dashboard analyzes an FMCG sales dataset to track revenue performance, product demand, and regional market distribution from 2023 to 2024.   The goal of this project was to transform raw sales data into interactive, actionable insights for stakeholders to optimize inventory and marketing strategies.  ## 🛠️ Key Features & Visualizations * **Executive Summary KPIs:** Real-time tracking of total sales volume ($27.99M) and total boxes shipped (2M). * **Time-Series Analysis:** A line chart displaying sales trends across Years, Quarters, and Months to identify seasonal demand shocks. * **Product Performance Analytics:** A dynamic horizontal bar chart ranking top-selling products (e.g., Orange Choco, Drinking Coco) based on order count and total amount. * **Geographical Distribution:** A breakdown of market share using a donut chart, highlighting major revenue drivers across the UK, USA, India, and Canada. * **Salesperson Leaderboard:** A detailed matrix visual ranking sales representatives alongside data bars for quick performance benchmarking. * **Interactive Slicers:** Date range sliders allowing users to filter the entire dashboard dynamically.  ## 🧠 Skills Demonstrated * Data Visualization & Dashboard Design * UI/UX Best Practices in Power BI (Clean grid layout, intentional color palettes) * Interactive Filtering & Slicing * Data Storytelling  ## 📂 How to View 1. Download the `.pbix` file from this repository. 2. Open it using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
